# Memory Leak

🎮 **Memory Leak** é um jogo desenvolvido na **Unity utilizando C#**, focado em exploração, suspense e narrativa ambiental.

## 🧠 Sobre o jogo

O jogador explora um ambiente misterioso enquanto coleta pistas e interage com o mundo ao redor. Conforme avança, acontecimentos estranhos começam a surgir, levantando dúvidas sobre o que é real e o que pode estar acontecendo com o personagem.

A experiência do jogo é construída por meio da exploração, elementos visuais e mecânicas que incentivam o jogador a descobrir gradualmente a história.
